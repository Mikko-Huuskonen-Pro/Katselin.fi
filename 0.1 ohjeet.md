# Katselin.fi – Toteutusohje v0.1

## Tavoite

Rakenna erittäin kevyt ja nopea staattinen verkkosivusto.

Sivusto toimii:

- GitHub Pagesissa
- ilman JavaScript-riippuvuuksia
- ilman build-työkaluja
- pelkällä HTML5 + CSS3:lla

Sivuston tulee olla helposti luettava sekä ihmisille että tekoälyille.

---

# Teknologiat

- HTML5
- CSS3
- Ei frameworkeja
- Ei Bootstrapia
- Ei Reactia
- Ei Tailwindia
- Ei Node.js:ää
- Ei build-vaihetta

---

# Ulkoasu

Teema:

- tumma tausta (#000000)
- logon sininen päävärinä
- paljon tyhjää tilaa
- selkeä typografia
- mobile first

Tyyli:

- rauhallinen
- moderni
- luotettava
- merellinen

Ei animaatioita ensimmäisessä versiossa.
