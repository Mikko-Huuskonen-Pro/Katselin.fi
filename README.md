# Katselin.fi

**The official website and documentation for Katselin.**

This repository contains the public documentation, user guides and developer documentation for the Katselin browser.

## Contents

- 🌊 Project philosophy
- 💡 Majakka (User Guide)
- ⚓ Varustamo
- 🧩 Application API
- 📦 Application manifest specification
- 🏗 Architecture
- 🔒 Security model

## Purpose

This repository is intended to be readable by both humans and AI assistants.

Everything needed to understand, use and develop applications for Katselin is documented here.

The official Varustamo catalog, whitelists and signing infrastructure are maintained separately.
