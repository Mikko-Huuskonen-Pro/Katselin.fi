Feedback och felrapporter

Berätta om ett fel eller föreslå en förbättring. Knappen öppnar GitHub där rapporten sparas i det här repot.

Så fungerar det

  1. Fyll i formuläret nedan.
  2. Knappen öppnar GitHub med ett förifyllt ärende.
  3. Logga in på GitHub (gratis konto räcker) och bekräfta.
  4. Rapporten syns i repots Issues.

Webbplatsen använder inga cookies eller spårning. Inget skickas förrän du bekräftar ärendet på GitHub.

Fel i webbläsaren

Krasch, sidor som inte laddas, sökning i Hamnen eller annat programfel.

Skriv inte lösenord, personnummer eller bankuppgifter.

Säkerhet och integritet

Sårbarheter rapporteras inte som offentliga ärenden — säkerhetsanvisning (på finska).

Formuläret samlar inga uppgifter själv. När du bekräftar på GitHub blir ärendet offentligt. Skriv inte personuppgifter du inte vill publicera. GitHub behandlar data i USA — integritetspolicy.
